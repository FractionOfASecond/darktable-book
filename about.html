About

This project was started with the goal of making a better manual for darktable. Later on, taking into account comments from the community, we decided to change this project from a manual to a book.

This book is written by beginners for beginners. It drives the reader through a smooth learning process, starting with a simple tutorial first, then exploring the many functionalities provided by the program and finally explaining the most common processing (workflows) and the concepts behind them.
The book is written using Darktable v 8.0, but most of the concepts and descriptions of the functionality, user interface and workflows are valid of earlier versions of the program.
